## 🧩 OOP Concepts Implemented

> [!TIP]
> This project is a great showcase of how real-world data can be modeled using C++ classes.

*   **Encapsulation:** Protecting user health data within private class members.
*   **Inheritance:** Using a base `Exercise` class to derive specific types like `Strength` and `Cardio`.
*   **Polymorphism:** Utilizing virtual functions to calculate calories burned across different workout types.
